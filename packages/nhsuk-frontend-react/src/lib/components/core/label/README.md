# Label

## Basic Usage

```tsx
import { Label } from 'nhsuk-frontend-react'

const Component = () => {
  return <Label size="s">First name</Label>
}
```

## Custom Props

| name              | type      | default value |
| ----------------- | --------- | ------------- |
| `size`            | `Size`    | `undefined`   |
| `isisPageHeading` | `boolean` | `false`       |
